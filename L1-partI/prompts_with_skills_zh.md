# 使用 Skills 的提示词

## 第一部分

我附上了本周的营销广告活动数据。请分析数据，帮助我了解每个渠道的表现（漏斗分析和效率分析），以及我是否需要重新分配预算。我可以在各渠道之间转移最多 $10,000，且每个渠道的增加幅度不超过 15%。

请直接在聊天中呈现结果。

## 第二部分

创建一份 Excel 报告，包含以下内容：
* 执行摘要工作表，包含关键发现和预算重新分配表
* 漏斗分析工作表
* 效率分析工作表
* 原始数据工作表

使用颜色编码：绿色表示良好，红色表示不佳。


**注意**：在拍摄过程中，我们获得的 Excel 表格不包含公式或工作表之间的引用。如果您希望 Excel 表格包含公式而不仅仅是数值，可以在第二部分的提示词中添加以下内容：

- 请确保在 Excel 表格中包含公式和引用。

由于 Claude.ai 中的预览功能仅显示电子表格的简化视图，可能无法呈现所有 Excel 格式、公式或多个工作表。
