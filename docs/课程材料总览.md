要访问每节课的文件、参考资料和阅读笔记，您可以点击每个视频下方的 **查看材料**。所有材料均可在此 <a href="https://github.com/https-deeplearning-ai/sc-agent-skills-files/tree/main" target="_blank">代码仓库</a> 中获取。
