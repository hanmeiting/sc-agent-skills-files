## 📚 分阶段学习说明

### 🔰 第一阶段：理解基础概念（L1 + L2）

**目标：** 理解 Agent Skills 是什么、为什么需要它、以及它如何与其他 AI 工具协作。

| 步骤 | 文档 | 核心内容 | 预计时间 |
|------|------|---------|---------|
| 1 | [L1 第一部分](./L1-第一部分_为什么需要Agent-Skills.md) | 观看演示：用 Skills 自动化营销报告分析 | 30分钟 |
| 2 | [L1 第二部分](./L1-第二部分_为什么需要Agent-Skills.md) | 理解 Skills 的原理、结构和新范式 | 45分钟 |
| 3 | [L2](./L2_Skills与工具MCP和子Agent的比较.md) | 辨析 Skills vs MCP vs Tools vs 子 Agent | 30分钟 |

**L1 第一阶段完成标志：**
- ✅ 理解 Skills 解决了什么痛点
- ✅ 能说出 Skills 的文件夹结构（SKILL.md + 可选目录）
- ✅ 区分 Skills 与 MCP、Tools 的不同职责

---

### 🔧 第二阶段：动手体验（L3）

**目标：** 使用现有的预构建技能，感受 Skills 在真实场景中的工作方式。

| 步骤 | 文档 | 核心内容 | 预计时间 |
|------|------|---------|---------|
| 4 | [L3](./L3_探索预构建的Skills.md) | 探索和使用 Anthropic 提供的预构建 Skills | 60分钟 |

**本阶段练习：**
- 🛠️ 动手试验营销技能（Marketing Skill）与 BigQuery/SQLite 集成
- 🎨 体验品牌指南技能（Brand Guidelines Skill）
- 🔗 体验组合多个 Skills 完成完整工作流

**L3 阶段完成标志：**
- ✅ 成功运行至少一个预构建 Skill
- ✅ 理解 Skill 的渐进式披露机制

---

### 🏗️ 第三阶段：创建自己的 Skills（L4）

**目标：** 掌握从零开始创建高质量自定义技能的方法。

| 步骤 | 文档 | 核心内容 | 预计时间 |
|------|------|---------|---------|
| 5 | [L4](./L4_创建自定义Skills.md) | 学习 SKILL.md 结构、最佳实践与评估方法 | 90分钟 |

**本阶段关键知识点：**
- 📝 SKILL.md 的 YAML 前置内容必填/可选字段
- 📁 `/assets`、`/references`、`/scripts` 目录的用途
- 📊 三种自由度等级的选择
- 🧪 如何定义单元测试验证技能效果

**L4 阶段完成标志：**
- ✅ 能独立编写一个完整的 SKILL.md 文件
- ✅ 理解何时使用高/中/低自由度指令
- ✅ 能为技能编写基本测试用例

---

## 🔑 核心概念关系图

```
                    ┌──────────────────────────────┐
                    │         Agent Skills         │
                    │  (技能 = 文件夹 + SKILL.md)   │
                    └──────────────┬───────────────┘
                                   │ 提供专业知识
                    ┌──────────────┼───────────────┐
                    ▼              ▼               ▼
             ┌─────────┐   ┌─────────────┐  ┌──────────┐
             │  主Agent │   │   子 Agent  │  │  子Agent  │
             │  (主控)  │   │ (独立任务A) │  │(独立任务B)│
             └────┬────┘   └─────────────┘  └──────────┘
                  │
         ┌────────┼────────┐
         ▼        ▼        ▼
    ┌────────┐ ┌──────┐ ┌──────────┐
    │  MCP   │ │ 工具 │ │  文件系统 │
    │(数据源)│ │(能力)│ │ (代码执行)│
    └────────┘ └──────┘ └──────────┘

Skills = 告知"怎么做"
MCP  = 提供"数据访问"
Tools = 提供"执行能力"
子Agent = "独立工作者"
```
